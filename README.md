# PepeCoin Calculator
PepeCoin Calculator is a web-based tool for PepeCoin (PEP) holders to calculate the value of their holdings. The app allows users to input the number of shares they own and dynamically adjust the price per coin to see their total asset value. It also includes links to official PepeCoin resources.

Features

- Calculate the total value of your PepeCoin holdings.
- Dynamically adjust the price per coin using a slider.
- Persistent storage of the number of shares owned via localStorage.
